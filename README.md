# TaskList
Описание: Программа для контроля ежедневных задач.

## Бизнес логика:
- возможность добавления задачи и последующее его редактирование
- вход по паролю
- корректная работа клавиш(Enter, Tab)
- поле редактирования задачи

## Описание задачи:
- название
- описание
- время работы
- приоритет
- статус
